Configuration Reference

Required Parameters

Parameter Description Example
data-book-id Unique identifier for booking system "hotel123"
data-type "hotel" or "chain" "chain"

Layout & Appearance

Parameter Options Default
data-layout "inline", "column", "expand" "inline"
data-theme "light", "dark" "light"

Currency & Localization

Parameter Examples Notes
data-currency "EUR", "USD", "GBP", "JPY" Controls price display format
data-locale "es-ES", "en-US", "en-GB" Controls language and date formats

Hotel Selection

Parameter Description Example
data-hotel Preselect specific hotel by ID "playa-garden-selection-hotel-spa"

Date Constraints

Parameter Format Example
data-min-date YYYY-MM-DD "2024-07-01"
data-max-date YYYY-MM-DD "2024-12-31"

Occupancy Limits

Parameter Description Default
data-max-rooms Maximum number of rooms 5
data-max-adults Maximum adults per room 4
data-max-children Maximum children per room 4
data-min-child-age Minimum child age 0
data-max-child-age Maximum child age 17
data-default-adults Default number of adults 2
data-default-children Default number of children 0

Other Options

Parameter Description Default
data-show-promo-code Show promo code field true
data-show-prices Show prices true
data-base-url Booking system URL Default system URL
